June 2018 - January 2019

Equiti Games

Wordpress Development, Automation

Overview

I was brought onto the Equiti Games team by the social media personality Nixie Pixel, hired to build the customer-facing website for the company's upcoming ICO. Using incredible mockups provided by the team I built the site on Wordpress with full CSS animations and interactive elements.

Tech

To have readily available bandwidth we built both the website and app in docker containers. For the website there are two sub containers, the daily build and the static HTML build. This was done to allow marketing / non-IT staff to have access to the Wordpress site and create content using visual builders and then once the ICO traffic was expected to hit we could redeploy with Wordpress converted to static HTML. The static site held outbound hrefs leading to the ICO web app.

Languages / Frameworks / Platforms

  • Docker

  • Gatsby.js

  • mySQL

  • PHP

  • NGINX

  • Wordpress

Process

Equiti struggled early-on with project management. Most requests, revisions, docs, and assets were organized using Slack channels and Google Doc comments. Deadlines were always "in the next few days" and I would receive full section revisions weeks after the site had already launched. I hope that my postmortem I ran through with the project leads helped solidify for them that rigid processes and documentation are key to building tech that lasts.
